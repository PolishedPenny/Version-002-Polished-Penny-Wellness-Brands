# Vitality & Ancient Wisdom - Holistic Coaching Website

A professional, fully responsive website for holistic coaching services built with HTML, CSS, and JavaScript.

## 📁 Project Structure

```
vitality-website/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet
├── script.js           # JavaScript functionality
├── netlify.toml        # Netlify configuration

```

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #d4a574;
    --secondary-color: #8b6f47;
    --accent-color: #c9a876;
    --light-bg: #faf7f2;
    --dark-text: #2c2c2c;
    --light-text: #666666;
    --white: #ffffff;
}
```

## 🌐 Deployment to Netlify

### Option 1: Drag and Drop (Easiest)
1. Go to [netlify.com](https://www.netlify.com/)
2. Sign up or log in to your account
3. Drag and drop the entire `vitality-website` folder onto the Netlify dashboard
4. Your site will be live instantly!

### Option 2: GitHub Integration
1. Push this project to a GitHub repository
2. Connect your GitHub account to Netlify
3. Select this repository
4. Netlify will automatically deploy on every push

### Option 3: Netlify CLI
```bash
npm install -g netlify-cli
netlify deploy
```

## ⚡ Performance Tips

- Optimize images before uploading (use tools like TinyPNG or Squoosh)
- Keep image file sizes under 500KB for best performance
- Use modern image formats (WebP, JPEG)
- Netlify automatically caches CSS, JS, and image files
